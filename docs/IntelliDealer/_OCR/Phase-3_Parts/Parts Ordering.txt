           Field               Description
            Order              The parts order number.
            Number

            Vendor             The name of the part's vendor for this order.

            Comment Any comments related to the parts order.

            Type               The type of order.

            Line               The number of line items on the selected part
            Items              order.

            Status             The status of the parts order:

                                         On Order—the part order is currently on
                                         order, no parts can be added to the order.

           Type Options

            MD        Machine Down

            FI        Fill In

          Click on a vendor name on the Vendor field to open the Vendor

          Move the cursor over the Ordering Options link located at the
          top right of the screen, and select one of these options from the
          drop down list:

                                                     purchase order from the purchase
