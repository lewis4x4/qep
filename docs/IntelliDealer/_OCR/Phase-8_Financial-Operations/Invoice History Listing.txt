           Searching Invoice History

                    If you enter an Invoice Number, executing a search also
                    scans Integrated Rental Details for a matching
                    invoiceNumber using a starts-with search.

                    If you enter a Contract Number, executing a search also
                    scans:

                             Invoice History table for string constant $GB +
                             contractNumber

                             Integrated Rental Details table for matching
                             contractNumber using a starts-with search

                    If you enter your search into the Full Text Search,
                    executing a search also scans Invoice History Details for
                    matching lineDetails

           The fields on the Invoice History listing screen are:

            Field                Description

            Invoice              The invoice number or system reference number.
            Number
                                 The first digit of an invoice indicates the system
                                 from which the invoice was generated, as
                                 specified in the General Invoice First Digit . The
                                 Invoice 1st Digits are configured by location and
                                 can include special characters, any number from
                                 0-9 and any letter from A-Z, except for Q, L and R
                                 which are reserved for sales order invoice values.

            Location             The location where the invoice was processed.

            Type                 The invoice type.

            Customer The customer's name.
            Name

            Reference The invoice reference number.
            Number

            PO                   The purchase order number.
            Number

            Invoice              The date the transaction was invoiced.
            Date

            Invoice              The total charge for all billable items on the
            Amount               invoice.
