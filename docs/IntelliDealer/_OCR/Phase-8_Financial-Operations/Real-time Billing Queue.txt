           Security Prerequisites

           You can enable real-time billing for a location, or for only a few
           users at a time, allowing you to control the roll out of this feature
           in your dealership. For instructions see Real-time Billing
           System.

           The fields on the Real-time Billing Queue screen are:

            Field                 Description
            Reference             A number uniquely identifying the billing.
            Number
                                  For successfully processed invoices, this number
                                  is a hyperlink that opens the Data Portal
                                  displaying the related reports.

                                  For Rental invoices, hover over the hyper link and
                                  chose one of:
                                            Edit Contract—opens the rental contract
                                            Details tab so you can modify it.
                                            View Billing Reports—opens the Data
                                            Portal displaying the related reports.

            Location              The location that originated the billing.

            Type                  The type of billing.

            Status                The status of the billing. Can be one of:
                                            Pending—the billing is in the queue awaiting
                                            processing.
                                            Processing—the billing is currently
                                            processing.
                                            Failed—the billing failed. An error message
                                            is displayed.
                                            Completed—the billing completed
                                            successfully. The reference number is a link
                                            to the resulting reports.
