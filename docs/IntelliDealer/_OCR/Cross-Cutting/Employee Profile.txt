           The fields on the Employee Profile screen are:

            Field               Description
            Employee The employee number. The employee number can
            Number be up to five digits long consisting of alphanumeric
                     or trailing blank characters.

            Name                The name of the employee.

            Class               The class of the employee. This field determines
                                the G/L expense account which the employee's
                                wages are debited. Valid class codes are set up on
                                the Classes screen.

            Category A three character user-defined employee category.

            Profit              The profit center or department where the
            Center              employee's wages are charged.

            Pay Type The employee's pay type.
