          Screen Description

          If an image was added to the selected customer profile (as
          specified on the Multimedia tab), and the image was assigned
          the Portal classification on the Edit Link/ Add Link screen, it is
          displayed here.

          You can search for a customer portal by entering a customer
          number. If the customer has notifications, a link appears next to
          the customer number. Click the link to see the notifications. See
          Customer Notifications.

          The fields on the Customer Portal screen are:

           Field                    Description
            Status                  The customer's status.

            Name                    The customer's name.

            Address                 The customer's address.

            Zip/Postal              The customer's zip/postal code.
            Code

            Contact                 The name of the customer's primary contact.

            Phone                   The customer's phone number.
            Number

            Cell Number The customer's cell phone number.

            E-mail                  The customer's e-mail address.
            Address

            Salesperson The name of salesperson associated to the
                        customer.

            Comment                 A one line comment that is loaded on the
                                    customer's profile.
