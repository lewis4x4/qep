          Move the cursor over a Contract Number and select one of
          these options from the pop-up menu:

                    Edit - to edit a rental contract.

                    Print - to print a rental contract.

                    Close - to hard-close a rental contract. Hard-closing a
                    rental contract deletes the contract and any open invoices
                    associated with the contract, regardless of whether there
                    are pending invoices or unreturned units on the contract.


                                   Note: You cannot hard-close a rental contract
                                   currently in use by another user.


                    Return - to return a rental contract using one of the rental
                    return options Returned, Off Rent, Hold Unit,
                    Exchanged, or Partial Return.
