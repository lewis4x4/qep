             The fields on the Data Miner screen are:

             Field                             Description

             Query Name                        The name of the query.

             Description                       A brief description of the query.

             Created by                        The user who created the query.

             Date Created                      The date the query was created.

             Date Last Modified The date the query was last modified.

             Date Last Run                     The date the query was last run.


             Hover over a query name and choose one of these options:

                   Edit- edit the query.

                   Copy- copy the query.

                   Run- run the query.
