## How It Works (Under the Hood)

### Three Layers of Intelligence

```
┌──────────────────────────────────────────────┐
│           LAYER 3: AI ACTIONS                │
│   Create tasks, draft emails, move deals,    │
│   generate prep sheets — from conversation   │
├──────────────────────────────────────────────┤
│           LAYER 2: SMART TOOLS               │
│   20+ function-calling tools that query      │
│   live structured data on demand             │
├──────────────────────────────────────────────┤
│           LAYER 1: SEMANTIC SEARCH           │
│   Vector embeddings + keyword search across  │
│   all 16 data sources simultaneously         │
└──────────────────────────────────────────────┘
```

**Layer 1 — Semantic Search:** Every CRM record and document is converted into a mathematical representation (vector embedding) that captures its *meaning*, not just keywords. When you ask "What excavators do we have for forestry work?", it finds relevant equipment even if the word "forestry" doesn't appear — because it understands the concept.

**Layer 2 — Smart Tools:** The AI has 20+ specialized tools it can call in real-time: search contacts, query pipeline data, pull financing rates, get market valuations, check competitor pricing, analyze voice note trends, and more. It decides which tools to use based on your question.

**Layer 3 — AI Actions:** The assistant can go beyond answering questions — it can *do things*. Create follow-up tasks, log activities, draft emails, move deal stages, generate customer prep sheets — all from natural language.

### 2. Voice Note Intelligence

When a rep records a field note, the system doesn't just transcribe it. It:

- **Extracts structured data** — contact name, company, equipment interest, deal stage, budget, urgency, financing preference, competitors mentioned, and 40+ other fields
- **Auto-links to CRM records** — fuzzy-matches mentioned names to existing contacts and companies
- **Detects sentiment** — flags frustrated or skeptical customers
- **Tracks competitor mentions** — every time a competitor is named, it's logged for competitive intelligence
- **Flags for manager attention** — if the AI detects a deal that needs escalation

All of this happens automatically, in the background, every time a voice note is submitted.

**What this means:** A rep records a 90-second voice note after a site visit. Within seconds, the system has:
- Created a CRM activity with the full transcript
- Extracted the customer's equipment interest and budget range
- Linked it to the correct company and deal
- Flagged that a competitor was mentioned
- Noted the customer seemed frustrated about delivery timelines
- Marked it for manager review

No manual data entry. No forgetting to log the call. It just happens.

---

### 3. Morning Briefing Agent

Every morning, each team member can get a **personalized AI-generated briefing** that includes:

- **Pipeline snapshot** — total open deals, pipeline value
- **Priority actions** — the most important things to do today
- **Deals to watch** — closing soon or needing attention
- **Overdue follow-ups** — flagged urgently
- **Quick wins** — easy actions that move deals forward

The briefing is generated fresh each day based on live CRM data. It shows automatically when you open the Knowledge Base.

**What this means:** Your reps start every day knowing exactly what to focus on, without a manager having to tell them.

---

### 4. Anomaly Detection & Risk Alerts

The system continuously monitors your CRM for problems:

| Alert Type | What It Detects |
|---|---|
| **Stalling Deals** | No activity on a deal in 7+ days |
| **Overdue Follow-ups** | Past the scheduled follow-up date |
| **Activity Gaps** | Reps with no logged activity in 3+ days |
| **Pipeline Risk** | Early-stage deals with close dates coming up fast |

These alerts are available to the chat assistant. Ask *"What needs attention?"* or *"Are any deals stalling?"* and you get an immediate, prioritized answer.

**What this means:** Problems don't hide in spreadsheets. The system surfaces them before they become lost revenue.

### 5. Deal Coaching

The AI analyzes your historical win/loss patterns and coaches reps on active deals:

- Compares deal age to average winning deal duration
- Flags when activity has dropped below the success threshold
- Warns when deal stage and close date don't align
- Shows top reasons similar deals were lost
- Provides win rate benchmarks

Ask *"How should I approach the ABC deal?"* and the AI gives specific, data-backed coaching.

**What this means:** Every rep gets coached like they have a dedicated sales manager watching every deal.

## Security & Access Control

- **Row Level Security (RLS)** on every table — reps only see their own data, managers see their team's
- **Role-based policies** — admin, manager, owner, and rep access levels
- **No data leakage** — the AI can only access what the authenticated user's role permits
- **Service role isolation** — background processes use separate credentials
- **Encrypted integrations** — HubSpot tokens encrypted at rest
